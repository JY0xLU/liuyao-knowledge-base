# 古籍阅读索引

本索引用来把经典资料变成可执行的学习路线。它不复制古籍全文，而是记录每部书在六爻知识库中的用途、优先章节、可抽取规则和后续读书任务。

## 1. 阅读原则

古籍阅读不要一上来逐字背诵。更有效的顺序是：

1. 先用现代教程掌握术语和装卦流程。
2. 再读经典目录，知道每章解决什么问题。
3. 每次只抽取一种规则，例如只看「旬空」或只看「伏神」。
4. 把规则写成规则卡，标明来源、适用条件、例外。
5. 再用卦例验证规则，而不是把古籍句子直接当成绝对定律。

## 2. 《卜筮正宗》

### 定位

《卜筮正宗》适合当作系统框架书。它强调用神、原神、忌神、仇神、飞伏神、进退神、反吟、伏吟、旬空、月破等为卦内纲领，并把这些纲领整理成「十八论」。

| 模块 | 知识库用途 | 输出 |

## 3. 《增删卜易》

### 定位

《增删卜易》适合做案例训练书。它的价值在于大量分类卦例和经验辨析，而不是只背单句规则。

### 阅读方法

1. 按专题读，不按页码硬读。
2. 每读一例，只记录问题、用神、动爻、月日、结论、反馈。
3. 把同类问题归在一起，例如财、婚、病、官、行人、失物。
4. 对照《卜筮正宗》的规则卡，看是否一致。
5. 对不一致处保留为「争议规则」，不要立即合并。

### 重点抽取字段

| 字段 | 说明 |
|---|---|
| topic | 财、婚、病、官、文书、失物、行人等 |
| question | 原问题，尽量保留具体语境 |
| main_god | 主用神 |
| secondary_gods | 辅助用神 |
| month_day | 月建、日辰、旬空 |
| moving_lines | 发动爻 |
| transformations | 回头生克、化进退、化空破墓 |
| judgment | 古籍判断 |
| outcome | 已知反馈 |
| extracted_rule | 可复用规则 |
| conflict_note | 与其他规则的冲突或限制 |

## 4. 《黄金策》

### 定位

《黄金策》适合作为分类断法和总纲资料。它覆盖天时、年时、国朝、征战、身命、婚姻、产育、病症、医药、种作、求名、仕宦、求财、家宅、坟墓、词讼、逃亡、失脱、出行、行人、舟船等主题。

### 使用方式

- 用作专题断法的目录来源。
- 用作象义扩展，不直接覆盖用神主线。
- 与《卜筮正宗》的十八论交叉核对。
- 与《增删卜易》的案例结果交叉验证。

### 专题索引建议

| 专题 | 对应现代问题 |
|---|---|
| 天时 | 天气、环境变化、短期外部条件 |
| 身命 | 个人状态、长期趋势，但不宜泛问 |
| 婚姻 | 感情、婚恋、复合、配偶 |
| 病症、医药 | 健康、治疗、检查；必须以医疗为准 |
| 求名、仕宦 | 考试、求职、升迁、资格 |
| 求财 | 收入、货款、投资、交易 |
| 家宅 | 房屋、租赁、家庭、居住环境 |
| 词讼 | 官司、投诉、争议 |
| 失脱 | 失物、遗失、被盗 |
| 出行、行人 | 行程、归期、远近消息 |

## 5. 《易隐》

### 定位

《易隐》适合做扩展阅读。它更适合在掌握基础后，用来补充古籍脉络和门类差异。

### 处理方式

- 先建立目录，不急于抽规则。
- 与《卜筮正宗》《增删卜易》同题对照。
- 对不熟悉术语先入术语表，不直接入规则库。

## 6. 规则抽取模板

每条从古籍抽出的规则都按以下格式写入 `data/rules.json`：

```json
{
  "id": "rule-example",
  "title": "规则标题",
  "layer": "use_god",
  "statement": "规则内容",
  "use_when": ["适用条件"],
  "evidence_fields": ["需要检查的卦盘字段"],
  "exceptions": ["例外或争议"],
  "source_refs": ["ctext-bushi-zhengzong"],
  "confidence": "A"
}
```
