## 复盘规则

复盘时不要只写「准」或「不准」。至少回答：

1. 用神有没有取错？
2. 有没有忽略伏神、空亡、月破？
3. 动爻传导链是否写完整？
4. 六神是否被误当成主判？
5. 应期是否和问题尺度匹配？
6. 结论是否过度确定？
