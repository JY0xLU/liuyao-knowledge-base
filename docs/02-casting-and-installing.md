# 起卦与装卦

## 1. 起卦前提

六爻最怕问题混杂。起卦前先写清楚：

- 问题是什么。
- 求测人是谁。
- 事情对象是谁或是什么。
- 时间范围是什么。
- 希望判断的是成败、趋势、应期、选择，还是原因。

错误示例：

- 「我以后会不会好？」范围过大。
- 「这份工作和那个人还有钱财如何？」一卦多事。
- 「我想让它吉，看看卦怎么说。」意念不清。

更好的问法：

- 「我本周五面试这家公司，能否进入下一轮？」
- 「我和某人未来三个月是否会恢复联系？」
- 「这笔款项在 30 天内能否到账？」

## 2. 铜钱起卦

常见做法是三枚铜钱摇六次，从下往上记爻。

一种常用记法：

| 结果 | 爻 | 名称 | 是否动 |
|---|---|---|---|
| 一背两面 | 阳爻 | 少阳 | 不动 |
| 两背一面 | 阴爻 | 少阴 | 不动 |
| 三背 | 老阳 | 阳动 | 动，变阴 |
| 三面 | 老阴 | 阴动 | 动，变阳 |

不同教材对正反、阴阳的约定可能不同。关键是起卦前固定约定，并且全程一致。

## 3. 本卦、变卦、动爻

本卦是起卦当下的结构。动爻是事情变化的发动点。变卦是动爻变化后的结果或趋势背景。

判断时不要把变卦当成「必然结局」。更稳妥的读法：

- 本卦：当前局面。
- 动爻：变化机制。
- 变爻：动爻变化后显出的新状态。
- 变卦：变化后的整体背景。

## 4. 装卦总步骤

标准化流程：

1. 画六爻：自下而上。
2. 分上下卦：初二三为内卦，四五上为外卦。
3. 得本卦名和变卦名。
4. 定卦宫。
5. 纳地支。
6. 按卦宫五行配六亲。
7. 安世应。
8. 安六神。
9. 查伏神。
10. 标月建、日辰、旬空。

## 5. 纳甲纳支简表

下表是常用浑天甲子纳支的底层记忆表。方向为自下而上。

| 经卦 | 五行 | 内卦三爻 | 外卦三爻 | 记忆 |
|---|---|---|---|---|
| 乾 | 金 | 子、寅、辰 | 午、申、戌 | 乾内甲子，外壬午 |
| 震 | 木 | 子、寅、辰 | 午、申、戌 | 震木庚子庚午 |
| 坎 | 水 | 寅、辰、午 | 申、戌、子 | 坎水戊寅戊申 |
| 艮 | 土 | 辰、午、申 | 戌、子、寅 | 艮土丙辰丙戌 |
| 坤 | 土 | 未、巳、卯 | 丑、亥、酉 | 坤土乙未癸丑 |
| 巽 | 木 | 丑、亥、酉 | 未、巳、卯 | 巽木辛丑辛未 |
| 离 | 火 | 卯、丑、亥 | 酉、未、巳 | 离火己卯己酉 |
| 兑 | 金 | 巳、卯、丑 | 亥、酉、未 | 兑金丁巳丁亥 |

现代排盘软件会自动完成这一步，但手工学习时必须知道它的来源。纳甲错，后面六亲、旺衰、应期都会错。

## 6. 定卦宫

六十四卦分属八宫。每宫以一个八纯卦为首，再依次生成一世、二世、三世、四世、五世、游魂、归魂。

世爻常用规则：

| 类型 | 世爻 |
|---|---|
| 八纯卦 | 上爻 |
| 一世卦 | 初爻 |
| 二世卦 | 二爻 |
| 三世卦 | 三爻 |
| 四世卦 | 四爻 |
| 五世卦 | 五爻 |
| 游魂卦 | 四爻 |
| 归魂卦 | 三爻 |

应爻与世爻隔两位：

- 世在初，应在四。
- 世在二，应在五。
- 世在三，应在上。
- 世在四，应在初。
- 世在五，应在二。
- 世在上，应在三。

## 7. 配六亲

以卦宫五行为「我」，不是以世爻五行为「我」。

| 与卦宫五行关系 | 六亲 |
|---|---|
| 同我 | 兄弟 |
| 我生 | 子孙 |
| 我克 | 妻财 |
| 克我 | 官鬼 |
| 生我 | 父母 |

例：某卦属乾宫，乾宫五行为金。

- 金爻为兄弟。
- 水爻为子孙。
- 木爻为妻财。
- 火爻为官鬼。
- 土爻为父母。

## 8. 安六神

六神从初爻起排，按日干决定初爻起哪一神，再顺排。

| 日干 | 初爻起 |
|---|---|
| 甲、乙 | 青龙 |
| 丙、丁 | 朱雀 |
| 戊 | 勾陈 |
| 己 | 螣蛇 |
| 庚、辛 | 白虎 |
| 壬、癸 | 玄武 |

顺序固定为：

```text
青龙 -> 朱雀 -> 勾陈 -> 螣蛇 -> 白虎 -> 玄武 -> 青龙
```

## 9. 伏神

伏神是本卦六爻中没有出现的某类六亲，借本宫首卦对应爻藏伏在某爻之下。

判断伏神要看：

- 伏神是否为用神。
- 飞神生伏还是克伏。
- 伏神是否得月日生扶。
- 伏神是否遇冲可出。
- 伏神是否空破墓绝。

伏神常见于「用神不上卦」的情况。不上卦不代表没有，可能是事情隐藏、未显、被压、隔层、需待时而出。

## 10. 装卦检查表

每次装完卦，逐项检查：

- 六爻自下而上是否写反。
- 老阳老阴是否正确变爻。
- 本卦和变卦是否正确。
- 卦宫是否正确。
- 世应位置是否正确。
- 地支是否按内外卦分别纳入。
- 六亲是否按卦宫五行配，不是按世爻配。
- 六神是否按日干从初爻起。
- 月建、日辰、旬空是否记录。
