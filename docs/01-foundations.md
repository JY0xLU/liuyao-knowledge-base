# 基础理论

## 1. 与《周易》的关系

六爻使用六画卦作为基本载体。每一卦由六个爻组成，从下往上排列。传统《周易》读法重视卦辞、爻辞和义理解释；六爻纳甲法则把卦爻放入干支五行系统，用月日、动变、生克、六亲来分析人事。

因此学习六爻要分清两层：

- 卦象层：乾、坤、震、巽、坎、离、艮、兑及六十四卦。
- 纳甲层：每一爻对应地支、五行、六亲、六神、世应等判断信息。

## 2. 八卦基础

| 卦 | 象 | 常用五行 | 常见象义 |
|---|---|---|---|
| 乾 | 天 | 金 | 父、领导、规则、刚健、高处 |
| 兑 | 泽 | 金 | 口舌、少女、悦、缺口、谈判 |
| 离 | 火 | 火 | 文书、眼目、光明、附丽、信息 |
| 震 | 雷 | 木 | 动、长男、惊动、开始、车马 |
| 巽 | 风 | 木 | 入、长女、传播、文书、柔顺 |
| 坎 | 水 | 水 | 险、陷、盗、病、流动 |
| 艮 | 山 | 土 | 止、少男、阻隔、门庭、坟墓 |
| 坤 | 地 | 土 | 母、众、承载、田土、包容 |

八卦是符号库，不是单一对应表。同一个卦象在不同问题里有不同取象，必须结合用神、世应和动变判断。

## 3. 五行

### 生克

| 关系 | 顺序 |
|---|---|
| 相生 | 木生火，火生土，土生金，金生水，水生木 |
| 相克 | 木克土，土克水，水克火，火克金，金克木 |

六爻判断中，生克不是抽象哲学，而是力量传导：

- 生用神：资源、帮助、成事条件。
- 克用神：阻力、伤害、破坏条件。
- 用神生世：事情来益我。
- 世生用神：我去投入、消耗。
- 用神克世：事情压我、制我。
- 世克用神：我能控制事情，但也可能过度消耗。

### 旺相休囚死

最常见判断先看月建。用神临月、得月生扶，通常有根；被月冲克，通常力弱。日辰用于近身作用，可以生克冲合卦爻，也可以填实旬空、冲动静爻。

简化记忆：

- 当令者旺。
- 我生者相。
- 生我者休。
- 克我者囚。
- 我克者死。

不同门派表述会有差异，实战时要回到「月令给季节大势，日辰给当下触发」这一核心。

## 4. 天干地支

### 地支五行

| 五行 | 地支 |
|---|---|
| 木 | 寅、卯 |
| 火 | 巳、午 |
| 金 | 申、酉 |
| 水 | 亥、子 |
| 土 | 辰、戌、丑、未 |

### 地支六合

| 合 | 常见意义 |
|---|---|
| 子丑合 | 牵连、黏住、暗合 |
| 寅亥合 | 生合、相吸 |
| 卯戌合 | 合中带克，视问题而定 |
| 辰酉合 | 金象、合同、技术 |
| 巳申合 | 合中带刑，易反复 |
| 午未合 | 火土相连，热象、近邻 |

### 地支六冲

| 冲 | 常见意义 |
|---|---|
| 子午冲 | 南北、水火、急变 |
| 丑未冲 | 土库相冲、田宅、脾胃、阻滞打开 |
| 寅申冲 | 路途、车辆、职位、筋骨 |
| 卯酉冲 | 门户、口舌、关系决裂 |
| 辰戌冲 | 墓库、文书、房产、旧事翻动 |
| 巳亥冲 | 信息、流动、隐情、病灾 |

合不一定吉，冲不一定凶。合可能是成事，也可能是绊住；冲可能是破坏，也可能是冲开阻碍。

## 5. 六亲

六亲不是现实亲属表，而是以卦宫五行为「我」，按五行生克关系定出的五类人事符号。

| 六亲 | 关系 | 核心象 |
|---|---|---|
| 兄弟 | 同我者 | 同辈、竞争、劫财、朋友、分耗 |
| 子孙 | 我生者 | 子女、产出、快乐、技术、药物、制官鬼 |
| 妻财 | 我克者 | 钱财、资源、妻妾、可支配对象 |
| 官鬼 | 克我者 | 官职、压力、疾病、灾祸、丈夫、规则 |
| 父母 | 生我者 | 文书、证件、房屋、长辈、保护、学习 |

六亲的象义必须随题目切换。例如官鬼在求职中可能代表职位，在问病中可能代表病邪，在问官司中可能代表官方压力。

## 6. 六神

六神是辅助取象层，不宜越过六亲、旺衰、动变成为主判。

| 六神 | 常见象 |
|---|---|
| 青龙 | 喜庆、饮食、文雅、财喜、贵气 |
| 朱雀 | 口舌、文书、消息、争辩、声音 |
| 勾陈 | 田土、迟滞、旧事、牵连、稳定 |
| 螣蛇 | 惊疑、虚幻、缠绕、梦境、反复 |
| 白虎 | 伤灾、刀兵、急迫、强硬、丧病 |
| 玄武 | 隐私、盗贼、欺瞒、暧昧、水险 |

例：妻财临玄武，不等于一定被骗。要看妻财是否为用神、是否旺、有无动变克合、世应关系如何。六神只能提供「这件事像什么」，不能单独证明「会发生什么」。

## 7. 世应

世爻代表求测者、自身立场、自己这一方。应爻代表对方、外部环境、事情对象或对立面。

常见观察：

- 世旺：自己有力、主动、能承受。
- 世弱：自己被动、体力/资源不足。
- 应旺：对方强、外部条件强。
- 世应相生：双方有流通。
- 世应相克：双方有冲突或控制。
- 世应相冲：关系不稳，事情变化大。
- 世应相合：关系黏连，成事或牵绊都可能。

## 8. 经典资料定位

推荐把传统资料分工阅读：

- 《黄金策》：看总纲、分类断法、火珠林法的综合表达。
- 《卜筮正宗》：看系统化辨伪、十八论、用神原忌仇、飞伏等纲领。
- 《增删卜易》：看分类卦例、经验辨析和案例拆解。
- 《易隐》：看更广泛的纳甲筮法材料和古籍脉络。

初学不要一开始就逐字硬读古籍。先用现代教程掌握术语，再回到古籍校正规则。
