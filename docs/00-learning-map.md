# 学习地图

## 1. 六爻是什么

六爻，也常被称为文王卦、纳甲筮法、火珠林法，是以六爻卦象为骨架，把阴阳、五行、干支、六亲、六神、世应、动变、月日旺衰等要素组合起来判断事情趋势的传统占筮体系。

它和只读《周易》卦爻辞的路径不同。六爻的重点不是单纯解释卦辞，而是把每一爻装入干支五行系统，再用生克制化、旺衰空破、动变冲合来形成判断链。

## 2. 学习目标分层

### 入门层：会看卦

目标是看懂一张排好的六爻盘：

- 哪个是本卦、变卦、动爻。
- 哪个是世爻、应爻。
- 哪些是六亲：父母、兄弟、子孙、妻财、官鬼。
- 六神如何排列：青龙、朱雀、勾陈、螣蛇、白虎、玄武。
- 月建、日辰、旬空、月破、日破在盘里对应什么。

### 基础层：会装卦

目标是从一次起卦结果手工装出完整盘面：

- 三枚铜钱或数字起卦得到六爻阴阳。
- 标出老阳、老阴并生成变卦。
- 定卦宫。
- 纳地支。
- 配六亲。
- 安世应。
- 安六神。
- 查伏神。

### 判断层：会取用神

目标是把问题翻译为卦中焦点：

- 问财看妻财，兼看子孙生财、兄弟劫财、官鬼压力。
- 问官职事业看官鬼，兼看父母文书、妻财资源、子孙制官。
- 问感情婚姻，男问多取妻财，女问多取官鬼，同时看世应关系、合冲、动变。
- 问病看官鬼、世爻、用神受克、子孙制鬼及医药父母象。
- 问考试文书看父母，兼看官鬼名位、子孙发挥、世爻状态。

### 进阶层：会断证据链

目标不是凭感觉下结论，而是写出证据：

- 用神是否出现，是否两现、多现、伏藏。
- 用神在月日是否旺、相、休、囚、死。
- 用神是否空、破、墓、绝。
- 动爻对用神是生、克、冲、合、墓、化进、化退、化回头生、化回头克。
- 世爻和应爻代表的双方关系是否相生、相克、相冲、相合。
- 原神是否有力，忌神是否发动，仇神是否介入。
- 应期从动爻、冲合、旬空填实、墓库打开、月日临值等处取。

## 4. 训练节奏

### 第 1 周：术语识别

每天只做一件事：打开排盘结果，指出世应、六亲、六神、动爻、变爻、用神候选。

验收标准：不看教程也能说出每个标签是什么意思。

### 第 2-3 周：手工装卦

每天手工装 3 个卦。允许用排盘工具核对，但先自己写。

验收标准：能解释每一步为什么这样装，而不是只得到结果。

### 第 4-6 周：取用神

把问题分类，写出主用神、辅助用神、忌神、原神。

验收标准：一个问题至少能写出「主线」和「旁证线」。

### 第 7-10 周：证据链断法

开始完整断卦，但每个结论都要有证据。

验收标准：结论中不出现「感觉」「应该」；每句话都能回指到卦中某个爻、月日、动变关系。

### 第 11 周以后：案例复盘

只收集已反馈结果的案例。先断，再对照反馈找错因。

常见错因：

- 用神取错。
- 忽略伏神。
- 忽略旬空或月破。
- 把六神当主判。
- 只看用神旺衰，不看动变传导。
- 没有区分「有象」和「有力」。

## 5. 最小可用断卦流程

1. 明确问题：一卦只问一事。
2. 记录时间：年、月、日、时，至少要月建和日辰。
3. 排出本卦、变卦、动爻。
4. 定世应：谁是自己，谁是对方或外部环境。
5. 取用神：把问题翻译成六亲。
6. 看用神是否出现：现、伏、两现、多现。
7. 看用神旺衰：月日、长生墓绝、生扶克泄耗。
8. 看动变：谁动，动向谁，化什么。
9. 看原忌仇：生用者是否有力，克用者是否发动。
10. 看特殊状态：空、破、墓、合、冲。
11. 写结论：先趋势，再条件，再时间。
12. 复盘：记录实际反馈。
