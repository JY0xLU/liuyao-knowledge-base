# 断卦框架

## 1. 一条主线

六爻断卦的主线可以压缩为一句话：

> 以问题定用神，以世应定关系，以月日定旺衰，以动变定变化，以原忌仇定成败条件，以冲合空破墓定时间和转折。

不要先看神煞，也不要先看卦名吉凶。卦名可以给背景，但多数实战判断必须回到爻。

## 2. 取用神

用神是所问事情在卦中的代表。取错用神，后面判断越细越错。

| 问题 | 主用神 | 辅助观察 |
|---|---|---|
| 钱财、收入、货款 | 妻财 | 子孙生财、兄弟劫财、父母合同 |
| 工作职位、官职 | 官鬼 | 父母文书、世爻状态、财生官 |
| 考试、证书、合同 | 父母 | 官鬼名位、子孙发挥、朱雀信息 |
| 感情，男问女 | 妻财 | 应爻、世应生克合冲、官鬼竞争 |
| 感情，女问男 | 官鬼 | 应爻、妻财竞争、父母阻隔 |
| 子女、作品、技术 | 子孙 | 父母约束、官鬼压力 |
| 疾病 | 官鬼 | 世爻身体、子孙药物、白虎病伤 |
| 失物 | 妻财或物象对应六亲 | 伏神、玄武、方位、内外卦 |
| 房屋、车、证件 | 父母 | 勾陈田宅、朱雀文书、世应权属 |
| 竞争、破财、朋友 | 兄弟 | 财爻受克、应爻同类 |

## 3. 世应关系

世应是人事关系的骨架。

| 关系 | 倾向 |
|---|---|
| 世生应 | 我主动投入，对方得益 |
| 应生世 | 对方帮我，外部条件向我 |
| 世克应 | 我能控制或压制对方 |
| 应克世 | 对方压我，外部压力大 |
| 世应相合 | 有联系、黏连、合作或牵绊 |
| 世应相冲 | 分离、冲突、变化、距离 |
| 世应同类 | 同行、同辈、竞争或同气 |

世旺不一定吉，应旺不一定凶。问求职时应旺可能公司强；问诉讼时应旺可能对方强；问合作时世应都旺且相生更稳。

## 4. 旺衰判断

### 先看月建

月建代表季节与大环境。用神得月建生扶，说明有根；被月建冲克，说明基础差。

重点：

- 临月：用神与月建同地支，力量大。
- 月生：月建生用神，资源足。
- 用神生月：用神泄气。
- 月克：大环境压制。
- 月冲：月破，基础受损。

### 再看日辰

日辰代表当下触发和近身力量。

重点：

- 日生用神：当下有助。
- 日克用神：当下受制。
- 日冲静爻：可能暗动。
- 日合用神：事情被合住或得到牵引。
- 日辰填实旬空：空亡可被填。

### 再看卦内

卦内看同党和敌党：

- 原神是否出现、是否发动、是否有力。
- 忌神是否发动、是否克用。
- 仇神是否克原神或生忌神。
- 用神是否被兄弟、官鬼、父母等关键爻牵制。

## 5. 空、破、墓、绝

### 旬空

旬空表示暂时不实、不到位、虚、缺席、失约、落空。空不等于永远没有。

常见转实条件：

- 出空。
- 值日。
- 被冲。
- 动而化实。
- 得月日强扶。

### 月破、日破

被月建冲为月破，通常比日破更重。日破偏当下冲散。

判断：

- 用神月破，多主基础受损。
- 忌神月破，阻力受损，反可为喜。
- 原神月破，帮助不足。

### 墓库

墓可理解为收纳、隐藏、入库、停滞、关闭。是否为凶，要看事情性质。

- 钱财入库：可能是存起来，也可能是拿不出。
- 病入墓：可能病势收敛，也可能闭塞难解。
- 人入墓：可能不动、隐藏、住院、被困。

冲墓、开库、值墓都可能成为应期。

## 6. 动爻和变爻

动爻是事件的发动点。静爻有象，动爻有力。

常见结构：

- 动生用：有助力。
- 动克用：有阻力。
- 用神发动：事情自己动起来。
- 忌神发动：麻烦主动来。
- 原神发动：帮助出现。
- 用神化进：趋势增强。
- 用神化退：趋势减弱。
- 化回头生：变化后反得生。
- 化回头克：变化后受克。
- 化空：动而不实。
- 化破：动后受损。

不要只看动爻本身，要看「动爻 -> 变爻 -> 用神/世应」的传导路径。

## 7. 原神、忌神、仇神

以用神为中心：

- 原神：生用神者。
- 忌神：克用神者。
- 仇神：克原神、或生忌神者。

例：用神为妻财木。

- 原神为子孙水。
- 忌神为兄弟金。
- 仇神为父母土，因为土克水，也生金。

判断时要看它们是否有力：

- 有力原神发动，常能救用。
- 有力忌神发动，常伤用。
- 忌神空破，阻力可能虚。
- 原神空破，帮助可能不到。

## 8. 应期

应期不要硬猜日期，要从卦中触发点取。

常见取法：

- 用神临值之日、月。
- 冲开合住之日。
- 填实旬空之日。
- 冲墓开库之日。
- 动爻或变爻对应地支之日。
- 世应冲合对应地支之日。
- 问短事取日时，问中期取月，问长期取年。

应期必须和事情尺度匹配。问「今天能否收到消息」不宜断到明年；问「房产交易」也不宜只盯小时。

## 9. 断语模板

可以按这个顺序写：

```text
问题：
用神：
世应：
用神状态：
原神/忌神/仇神：
动变：
特殊状态：
结论：
条件：
应期：
风险：
反馈：
```

结论示例：

```text
此卦主线不利于本周内到账。妻财为用，虽现于卦中，但受月冲又被兄弟发动克制；原神子孙不动且空，生财条件不足。若后续到财爻出空或兄弟受冲之日，可能出现部分进展，但当前证据更像延迟而非立即完成。
```
